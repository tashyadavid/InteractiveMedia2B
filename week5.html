Paint an Image

In this sketch, I explored how an image can be used like a paintbrush in p5.js. As I moved the mouse across the canvas, the image would duplicate and follow the cursor, creating a trail of visual texture. It reminded me of digital stamping or collage-making, every movement left a mark that built up a composition over time.

I used two images in this experiment. When the mouse is pressed, one image appears, and when it’s not, the other one is used. This simple switch made the interaction feel more dynamic and playful — almost like I was toggling between two brushes or layers.

This exercise taught me how versatile image() can be when used creatively. It also helped me understand how conditional statements (if/else) can be used to control interactivity.

Code Snippet


            var img, img2;
            
            function preload() {
              img = loadImage("data/img.jpeg");
              img2 = loadImage("data/logo.jpeg");
            }
            
            function setup() {
              createCanvas(windowWidth, windowHeight);
            }
            
            function draw() {
              if (mouseIsPressed) {
                image(img, mouseX - 50, mouseY - 50, 100, 100 / img.width * img.height);
              } else {
                image(img2, mouseX - 50, mouseY - 50, 100, 100 / img.width * img.height);
              }
            }

Text to Point

In this sketch, I used textToPoints() to break up the phrase Les Points into individual coordinates and drew circles at each point. As the mouse moves across the screen, the size of the circles changes. This gave the text a dynamic, pulsing feel.

It actually inspired the way I approached interactivity on my homepage, specifically how I mapped mouse position to control the volume of sound. I liked how subtle cursor movement could change the experience in real time.

Code Snippet:


let font;
let points = [];

function preload() {
  font = loadFont('https://cdnjs.cloudflare.com/ajax/libs/topcoat/0.8.0/font/SourceCodePro-Regular.otf');
}

function setup() {
  createCanvas(windowWidth, windowHeight);
  background(160, 0, 160);

  const fontSize = 144;
  const bounds = font.textBounds("Les Points", 0, 0, fontSize);

  const x = width / 2 - bounds.w / 2;
  const y = height / 2 + bounds.h / 2;

  points = font.textToPoints("Les Points", x, y, fontSize, {
    sampleFactor: 0.2,
  });
}

function draw() {
  background(160, 0, 160);
  let xMapped = map(mouseX, 0, width, 4, 32);

  for (let p of points) {
    noStroke();
    fill(160);
    circle(p.x, p.y, xMapped);
  }
}

Webcam Experiment

I played around with how a webcam could be used to add fun interactivity. Instead of just showing the live feed, I added a posterize filter to give it a more stylised, graphic look, kind of like a digital stencil effect. I later incorporated this idea into my homepage to make the experience feel more personal and playful.

Code Snippet:


var capture;

function setup() {
  createCanvas(windowWidth, windowHeight);
  capture = createCapture(VIDEO);
  capture.size(640, 480);
  capture.hide();
}

function draw() {
  background(255);
  image(capture, width / 2 - 320, height / 2 - 240, 640, 480);
  filter(POSTERIZE, 4);
}

function windowResized() {
  resizeCanvas(windowWidth, windowHeight);
}

Dot to Dot

This was another fun experiment using one of the code snippets from the week 5 module. I didn’t change much, but just playing around with it showed me how even a super simple sketch can add personality to a webpage. It tracks your mouse and adds a counter every time you press a key, it really brings the screen to life in a fun way.

Code Snippet:


var counter;

function setup() {
  createCanvas(windowWidth, windowHeight);
  background(255, 255, 255);
  counter = 0;
  textSize(20);
}

function draw() {
  if (mouseIsPressed) {
    strokeWeight(10);
    stroke(255, 0, 0);
    line(mouseX, mouseY, pmouseX, pmouseY);
  }
}

function keyPressed() {
  counter++;
  fill(0);
  circle(mouseX, mouseY, 5);
  text(counter, mouseX, mouseY);
}
